# WhatsApp Server

שרת WhatsApp Web.js לשליחת הודעות אוטומטיות.

## התקנה מקומית

```bash
npm install
npm start
```

השרת ירוץ על `http://localhost:3001`

## פריסה ל-Railway

Railway יזהה אוטומטית את תיקיית `server` ויבנה אותה.

אם Railway לא מזהה נכון:
1. ב-Railway Dashboard → Settings → Root Directory
2. הגדר ל: `server`
